# Hey, I'm Sri Balaji

Data engineer, originally from Chennai, now based in Buffalo, which has become my second home over the years. I moved to the US for grad school, and it took a couple years of studying and exploring before I realized how much I didn't know, and how much there is to learn in life, tech, and engineering. Been chasing that ever since.
